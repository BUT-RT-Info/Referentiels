Module officiel¶

Module dédié aux éléments synthétiques (extrait des tableaux de synthèse maintenu par le PACD) sur les ressources, les SAés, les Compétences et les ACS. Parmi ces éléments, les codes et les noms officiels ; des tags pour calcul d’information de répartition.

Les ressources sont décrites dans un fichier yaml/ressources.yml structuré par semestre et par code, de la forme :

S1:
  R101:
    nom: "Initiation aux réseaux informatiques"
    tags-thematiques: ["réseau", "métier"]

Idem pour les SAés, dans yaml/saes.yml de la forme :

S1:
  SAE11:
    nom: "..."
    tags-thematiques: ["métier"]
class officiel.ACs(fichieryaml)¶

Modélise une liste de acs lorsqu’elle est extraite d’un fichier yaml

class officiel.Competences(fichieryaml)¶

Modélise une liste de compétences lorsqu’elle est extraite d’un dictionnaire

class officiel.Officiel¶

Récupère et stocke des informations dites officielles, fournies par différents fichiers yaml (conçus manuellement).

Certaines traduisent les tableurs de synthèse du GT-BUT avec :

  • le nom des ressources (cf. ressources.yml)

  • le nom des saés (cf. saes.yml)

  • les compétences (cf. RT123.yml)

  • les apprentissages critiques (structurées par compétences) (cf. acs.yml)

D’autres donnent : les mots-clés, les abréviations propres à RT (cf. motscles.yml, abreviations.yml).

Fournit également différentes méthodes pour exploiter ces données officielles.

get_ressource_name_by_code(code)¶

Pour un code de ressource valide (existant dans DATA_RESSOURCES) et ce sans connaissance du semestre, fournit le nom officiel de la ressource

get_sae_name_by_code(code)¶

Pour un code de saé valide (existant dans DATA_SAES) et ce sans connaissance du semestre, fournit le nom officiel de la sae

get_sem_sae_by_code(code_sae)¶

Récupère le semestre de la sae d’après son code_sae

officiel.devine_code_by_nom_from_dict(champ, dico)¶

Partant d’une chaine de caractères décrivant une ressource, une SAé ou un ACS, détermine le code présent dans le dico officiel (dico à double entrée), Le dico officiel vient d’un .yml

officiel.get_DATA_ACS(repertoire='yaml/pn')¶

Récupère les informations des ACS, en extrayant les données du fichier yaml/competences.yml

officiel.get_DATA_RESSOURCES(repertoire='yaml/pn')¶

Récupère les informations sur les ressources (triées par semestre et par nom), en extrayant les données du fichier yaml/ressources.yml.

officiel.get_DATA_SAES(repertoire='yaml/pn')¶

Récupère les informations des SAés (triées par semestre et par nom), en extrayant les données du fichier yaml/saes.yml

officiel.get_code_from_nom_using_dict(nom, dico)¶

Récupère le code d’une ressource d’après son nom en utilisant les noms officiels des ressources du yaml si dico == DATA_RESSOURCES ; sinon fait de même avec les SAE

officiel.get_officiel_name_by_code_using_dict(code, dico)¶

Extrait un nom à partir d’un code (pour les RESSOURCES ou les SAES)

officiel.supprime_accent_espace(chaine)¶

Met une chaine en minuscule, supprime les accents, les ponctuations et les espaces
